# Testing React-Application

## Setup

```bash
npm install mocha react-addons-test-utils jsdom enzyme --save-dev
```

## `package.json`

```json
"scripts": {
  "test": "mocha",
  "test:watch": "npm test -- --watch",
}
```

## Running tests

```bash
npm test
npm run test:watch
```
